## üîß Essential Components

<Tabs>
<Tab title="üß† PayloadTransformer">
  **The Heart** - Transforms JSON payloads using metadata configuration
  
  **Key Methods**:
  - `transformToOrderAndOrderItems()` - Complex order processing
  - `transformPayloads()` - Generic entity transformation  
  - `applyTransformation()` - 15+ business rule types
  
  **Configuration**: SystemFieldMapping__mdt Custom Metadata Type
</Tab>

<Tab title="üéØ Integration Utilities">
  **The Muscle** - Applies complex business logic after transformation
  
  **ShopifyIntegrationUtilities**:
  - Order post-transformation rules
  - Multi-currency pricebook management
  - Product auto-creation logic
  
  **CegidIntegrationUtilities**:
  - Multi-entity API integration
  - Customer categorization (VIP/Employee/Press)
  - Territory and commission assignment
</Tab>

<Tab title="üìä MiddlewareAction__c">
  **The Memory** - Complete audit trail for every operation
  
  **Key Fields**:
  - `Status__c` - Completed/Pending/Failed
  - `Error__c` - Detailed error messages
  - `Payload__c` - Original request for debugging
  - `NbRetries__c` - Retry attempts remaining
</Tab>
</Tabs>

## üö® Quick Troubleshooting

<AccordionGroup>
<Accordion title="üõí Orders Not Appearing in Salesforce">
  **Quick Diagnosis**:
  
  <Steps>
  <Step title="Check Security">
    Review debug logs for HMAC signature validation failures
  </Step>
  <Step title="Query Errors">
    Check `MiddlewareAction__c` records for specific error details
  </Step>
  <Step title="Verify Webhooks">
    Confirm webhook secrets match between external system and Salesforce
  </Step>
  </Steps>
  
  **Quick Fix**: Update webhook configuration in external system
</Accordion>

<Accordion title="üí± Currency-Related Failures">
  **Symptoms**: "No Pricebook available for currency" errors
  
  **Quick Checks**:
  - Verify Pricebook2 exists for the specific currency
  - Confirm multi-currency is enabled in Salesforce org
  - Check Standard Pricebook entries exist for products
  
  **Quick Fix**: Create pricebook for new currency or implement fallback policy
</Accordion>

<Accordion title="üë• Duplicate Customer Accounts">
  **Symptoms**: Multiple accounts created for same customer
  
  **Root Causes**:
  - Customer email address changed in external system
  - External ID mapping configuration issues
  - Deduplication logic bypassed
  
  **Quick Fix**: Review customer matching strategy and merge duplicates manually
</Accordion>

<Accordion title="üè™ Products Not Creating Automatically">
  **Symptoms**: DML exceptions on Product2 insert operations
  
  **Common Issues**:
  - Duplicate ProductCode values from concurrent webhooks
  - Required field validation rule failures
  - Governor limits exceeded
  
  **Quick Fix**: Check for existing products before creation attempt
</Accordion>
</AccordionGroup>

## ‚öôÔ∏è Key Configuration

<Tabs>
<Tab title="üó∫Ô∏è SystemFieldMapping__mdt">
  **Purpose**: Maps external JSON fields to Salesforce fields
  
  **Essential Fields**:
  - `ExternalApiPath__c` - JSON path (e.g., `customer.email`, `lines[].quantity`)
  - `SalesforceField__c` - Target field (e.g., `PersonEmail`, `Quantity`)  
  - `TransformationLogic__c` - Business rule (e.g., `Concatenate`, `ShopifyStatusLogic`)
  - `IsActive__c` - Enable/disable specific mappings
  
  **Location**: Setup ‚Üí Custom Metadata Types ‚Üí SystemFieldMapping
</Tab>

<Tab title="üîê ExternalSystemCredential__mdt">
  **Purpose**: Secure storage of API credentials and endpoints
  
  **Key Fields**:
  - `BaseURL__c` - External system API base URL
  - `Token__c` - Authentication token/secret (encrypted)
  - `TenantID__c` - Multi-tenant identifier for Cegid
  
  **Location**: Setup ‚Üí Custom Metadata Types ‚Üí ExternalSystemCredential
</Tab>

<Tab title="üÜî SystemExternalIDs__mdt">  
  **Purpose**: Defines external ID field mapping per system and object
  
  **Configuration**:
  - `ExternalSystemName__c` - System identifier (e.g., 'Shopify Ami Paris')
  - `SalesforceObject__c` - Object name (e.g., 'Order', 'Account')
  - `ExternalIdName__c` - External ID field API name
  
  **Location**: Setup ‚Üí Custom Metadata Types ‚Üí SystemExternalIDs
</Tab>
</Tabs>
